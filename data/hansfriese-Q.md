## [L-03] We should use the latest verion of solidity
- https://github.com/PartyDAO/party-contracts-c4/blob/3896577b8f0fa16cba129dc2867aba786b730c1b/contracts/crowdfund/AuctionCrowdfund.sol#L2

Most contracts have `pragma solidity ^0.8;` and we should use the latest verion `0.8.15`.

Especially many functions use ABI-encodings in this protocol and `0.8.13` is known to have some issues related to ABI-encoding like explained [here](https://github.com/code-423n4/2022-06-putty-findings/issues/348).
